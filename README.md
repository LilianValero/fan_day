fan_day
=======

# Tapatalk XML-RPC testing tool

## Setup

Install Node.js with an [installer](http://nodejs.org/download), or if you have [Homebrew](http://brew.sh/):

```
brew install node
```

Then install Node packages by executing in the project root:

```
npm install
```

## Run

Run `main.js` with:

```
node main tottenham
```
